Volatility Drag Simulator: Arithmetic Return vs. Geometric (Compound) Return

Same 0.0% Expected Return. Different Compound Growth.

This simulator is not a portfolio tool and does not provide investment advice. It is a conceptual illustration of one mechanism: long-term investing is not only shaped by the expected return assumption, but also by the volatility path through which that return is earned.

The purpose is to make the difference between arithmetic return and geometric return visible. The model is illustrative, simplified, and not a forecast.

0.0%

The zero line is not neutral. Below it, volatility drag exceeds the arithmetic expected return and the approximate geometric return turns negative. At 0.0% expected return, the geometric return reaches 0% at 0.0% volatility.

Volatility drag approximation:  geometric return ≈ arithmetic return − ½ × volatility²
